# Harvest RNG - New Player Walkthrough

This walkthrough is the intended first-session path for a brand-new player. It assumes the default v1 settings: 250 starting coins, 3 unlocked plots, 50 coins per single roll, and 450 coins for a x10 roll.

## First Goal

Fill all 3 starting plots, harvest the first crops, then buy the first useful upgrade.

By the end of the first few minutes, the player should understand:

- Coins buy seed rolls.
- Seeds go into inventory before planting.
- Plots grow in real time.
- Harvesting pays coins.
- Coins should be reinvested into rolls, Luck, Harvest Speed, and more plots.

## Step-by-Step First Session

### 1. Join the farm

Look at the HUD first:

- Coins starts at 250.
- Gems starts at 0.
- Luck starts at 0 unless the player owns VIP Plot.
- Streak shows the current daily streak.

The farm starts with 3 unlocked plots. Locked plots can be bought later with coins.

### 2. Roll three seeds

Click `Roll (50 coins)` three times.

This spends 150 coins and leaves 100 coins in reserve. Each roll adds a seed to the inventory and shows a rarity reveal:

| Rarity | What it means |
|---|---|
| Common | Fast, low-value starter crops |
| Uncommon | Still quick, noticeably better payout |
| Rare | Strong early win, usually worth planting immediately |
| Epic+ | Big pull; plant it before leaving if it takes a long time |

Do not use `Roll x10` yet. It costs 450 coins, so it becomes a better option after the first few harvests.

### 3. Plant all starting plots

Click `Plant` on an empty plot, choose a seed from the picker, and repeat until all 3 plots are growing.

Good early priority:

1. Plant any Rare or better seed first.
2. Plant Uncommon seeds next.
3. Use Common seeds to keep every plot active.

Empty plots do not earn anything, so a full farm matters more than waiting for a perfect seed.

### 4. Watch the timers

Each planted plot shows a countdown. Early crops usually finish quickly:

| Rarity | Typical first-session wait |
|---|---|
| Common | 30-55 seconds |
| Uncommon | 75-115 seconds |
| Rare | About 3-5 minutes |
| Epic | About 10-15 minutes |
| Legendary | About 30-60 minutes |
| Mythic | About 2-4 hours |

When a crop is ready, the plot shows `Ready!` and the `Harvest` button appears.

### 5. Harvest and roll again

Click `Harvest` on each ready plot.

Harvesting:

- Adds coins to the wallet.
- Adds lifetime value to leaderboard progress.
- Clears the plot so another seed can be planted.

After the first harvest, roll enough seeds to refill empty plots. The core habit is simple: never leave an unlocked plot idle if there is a seed available.

### 6. Buy the first upgrade

Once the player reaches 200 coins, open `Upgrades` and buy `Luck Lv1`.

Luck Lv1 adds +5 Luck, slightly shifting roll odds away from Common and toward rarer tiers. It is the clearest first upgrade because it makes every future roll a little more exciting.

After that, choose based on play style:

| Choice | Best for |
|---|---|
| Luck upgrades | Players chasing Rare, Epic, Legendary, and Mythic pulls |
| Harvest Speed upgrades | Players actively watching plots and harvesting often |
| Unlock Plot 4 at 500 coins | Players who want more passive income capacity |

### 7. Save for Plot 4

The fourth plot costs 500 coins. Unlocking it is the first major farm expansion.

Once Plot 4 is unlocked, the player should repeat the same loop:

1. Roll seeds.
2. Keep every unlocked plot planted.
3. Harvest ready crops.
4. Reinvest into upgrades and plot unlocks.

## What To Do With Rare Pulls

Rare, Epic, Legendary, and Mythic seeds are the exciting moments. If the player gets one early:

- Plant it if a plot is open.
- If all plots are full, wait for the next ready crop and plant the rare seed after harvesting.
- Long-grow seeds are good before logging off, because they can mature while the player is away.

## Returning Player Routine

When coming back later:

1. Check for daily streak rewards.
2. Harvest any ready plots.
3. Replant empty plots.
4. Spend extra coins on upgrades or new plots.
5. Roll again when the farm needs seeds.

Auto-Farm owners can let ready plots harvest automatically every 3 seconds while they are in-game.

## Early Strategy Tips

- Keep plots full. Three planted Commons are better than one empty plot waiting for a better seed.
- Use Harvest Speed if you are actively playing.
- Use Luck if you mostly care about better rolls.
- Unlock new plots when rolls and upgrades feel affordable.
- Plant long timers before leaving the session.
- Check the leaderboard after a few harvests to see lifetime progress.

## First-Session Success Criteria

A new player has completed onboarding once they have:

- Rolled at least 3 seeds.
- Planted all 3 starting plots.
- Harvested at least 1 crop.
- Replanted an empty plot.
- Seen where upgrades and locked plots live.
- Understood that x10 rolls are a later convenience once they can afford 450 coins.

## Tester Notes

For a clean first-session test, start from a fresh save and verify:

- Player spawns with 250 coins and 3 unlocked plots.
- Three single rolls cost 150 coins total.
- Inventory picker opens from the plot `Plant` button.
- Plot timers reach `Ready!` at the expected time.
- `Harvest` adds coins and clears the plot.
- First Luck upgrade costs 200 coins.
- Plot 4 unlock costs 500 coins.
